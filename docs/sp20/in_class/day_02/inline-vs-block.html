Block elements

Block level elements take up the entire length of the container they're in. This is a <div>, so it takes up the whole width of the page, even if the written content doesn't.

Block level elements go onto the next line, and you can adjust their width and height.








Inline elements



inline elements only take up as much space as their content. inline elements will end up on the same line as other inline elements. this is an <a> tag.
