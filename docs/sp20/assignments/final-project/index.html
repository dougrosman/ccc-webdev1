Summary:

Create a website. That's it!*

*Ok there are few guidelines, but this is largely an open-ended project with some minimum technical requirements you have to include. I want you to use this final as an opportunity to make something that interests you. That could be:

  • A portfolio for your art/design/writing/whatever
  • A web tool, or a website with some specific purpose. Here are some examples:

  • A piece of web art (using the web as an artistic medium) or "artware" (a hybrid "art" and "software" website that does something kind of neat)
    • rrrgggbbb.com
    • pointerpointer.com
    • intotime.com
    • therevolvinginternet.com

Guidelines:

  1. Create an account on freecodecamp.org
    1. You need an account so you can record your progress, which you will need to get credit for this assignment.
    2. After you create your account, click the 'Update my account settings' button at the top of the freecodecamp.org/learn page.
    3. Create a username and click 'Save.'
    4. Scroll down to Privacy settings and set My Profile, My certifcations, My portfolio, and My time line to 'Public'
    5. At the top of the page, click 'Show me my public portfolio.'
    6. The public link to your profile will be something like freecodecamp.org/dougrosman. That's what you'll submit to Canvas to get credit.
  2. Complete Tutorials 1-18 of the Introduction to jQuery tutorial, which you'll find in the Front-end Libraries section.
    1. Each tutorial builds incrementally off the previous one, so I recommend trying to complete all of these in one or two sittings.

Tips/resources for getting started:

Watch lecture 01 here for a broad jQuery overview!

Upload the link to your freecodecamp public profile (e.g. freecodecamp.org/dougrosman) to Assignment 7a in Canvas

Finish all 18 tutorials and upload your link to Canvas before the deadline.
